📜 V14推理引擎规则说明

一、核心定位

V14是一个可解释推理引擎，而非传统占卜工具。 引擎基于内部定义的规则库进行推理，该规则库综合自传统易经典籍及迭代训练中确定的规则。 测试集由规则库自动生成，因此引擎输出与规则预设完全一致（100%内部一致性）。

⚠️ 重要说明
若与特定占卜网站或易学流派结果存在差异，属于规则定义差异，而非引擎错误。 V14的核心价值在于提供可解释、可组合、可验证的推理过程。

二、起卦规则

1. 输入映射

用户输入任意文本（数字、生日、电话等），引擎使用哈希函数映射到六十四卦。

hash(input) mod 64 → 卦象索引(0-63)

2. 动爻规则

变爻位置由卦象索引决定，确保确定性。

动爻位置 = 卦象索引 mod 6

3. 变卦计算

将指定爻位阴阳反转（阳→阴，阴→阳），得到变卦。

变卦 = change_gua(原卦索引, 动爻位置)

三、世应规则（京房八宫体系）

世应位置依据京房八宫卦世应排法，每卦均有固定的世应位置。

四、六亲规则

六亲依据卦宫五行与爻位五行生克关系确定。

关系说明六亲
爻五行 = 卦宫五行同我者兄弟
爻五行 = 生卦宫五行生我者父母
爻五行 = 卦宫所生我生者子孙
爻五行 = 卦宫所克克我者官鬼
爻五行 = 卦宫所不胜我克者妻财
六亲 = f(卦宫五行, 爻位五行)

五、吉凶判断规则

1. 五行相生

金→水 水→木 木→火 火→土 土→金

2. 五行相克

金→木 木→土 土→水 水→火 火→金

3. 吉凶判定

条件结果
变卦五行 = 本卦所生吉（进神）
变卦五行 = 本卦所克凶（退神）
变卦五行 = 本卦五行平（比和）
吉凶 = judge(本卦五行, 变卦五行)

六、卦象组成

1. 六爻排法

六爻由上卦（下三爻）和下卦（上三爻）组成。

上卦 = BAGUA[卦索引 / 8]
下卦 = BAGUA[卦索引 % 8]

七、确定性保证

V14引擎是确定性的，相同输入必然产生相同输出。

✓ 确定性设计
  • 不依赖时间戳
  • 不使用随机数
  • 所有规则均为纯函数
